# Creating a new realm

To create a new realm you must have admin account and be able to login.

![create-realm.png](create-realm.png)

1. Navigate to http://localhost:8080/admin/
2. In the realm drop down menu, press "Create Realm"
3. Insert realm name, set it as enabled and press "Create"
4. (optional) Change realm settings in http://localhost:8080/admin/master/console/#/REALM_NAME/realm-settings
